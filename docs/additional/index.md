- 標準手順: [AI活用の標準業務フロー（1枚）](../introduction/02-standard-workflow/)

- 実務例: [第8章：AI活用の具体的場面](../chapters/chapter08/)

## ケース末テンプレ

ケーススタディを読み終えたら、次の4点を毎回そろえると再利用しやすくなります。

- 入力データ（元資料/参照元）
- 生成プロンプト（版管理）
- CARE評価結果（修正理由）
- 最終成果物と承認ゲート

具体的な書式は [付録A：実践的なツール・テンプレート集](../appendices/appendix-a/) を参照してください。
