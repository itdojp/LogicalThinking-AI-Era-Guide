# 第1章 演習問題 解答・解説

## 演習1-1：AI活用の現状分析

### 解答例

**1. 現在使用しているAIツール**
- ChatGPT：企画書作成、アイデア出し、文章校正
- Microsoft Copilot：Office文書の作成支援、データ分析
- DeepL：翻訳業務、多言語コミュニケーション
- 文字起こしAI：会議議事録作成、インタビュー記録

**2. AI導入前後での業務変化**
- **作業時間の短縮**：文書作成時間が約50%削減
- **品質の向上**：多角的な視点からのアイデア創出が可能に
- **業務範囲の拡大**：翻訳など、これまで外注していた業務を内製化
- **思考時間の確保**：定型作業から解放され、戦略的思考に時間を割ける

**3. 感じている課題や困りごと**
- **出力の検証負担**：AIの回答が正確か判断する必要がある
- **指示の難しさ**：期待する結果を得るための適切な指示が分からない
- **依存度の管理**：AIに頼りすぎて自身の思考力が低下する不安
- **情報管理**：機密情報の取り扱いに関する不安

### 解説
この演習では、AI活用の現状を客観的に把握することが目的です。具体的なツール名と用途を明確にすることで、AI活用の実態が見えてきます。また、メリットだけでなく課題も認識することで、本書で学ぶ内容の必要性が理解できます。

## 演習1-2：論理的思考の必要場面

### 解答例

**場面1：問題解決の優先順位付け**
- **必要な思考**：問題の重要度と緊急度を評価する分析的思考
- **具体例**：複数のクレーム対応で、影響範囲と深刻度から対応順序を決定

**場面2：提案書の作成**
- **必要な思考**：課題→原因→解決策の論理的な流れを構築する構造化思考
- **具体例**：売上改善提案で、データ分析→課題特定→施策立案の流れを明確化

**場面3：会議での意見集約**
- **必要な思考**：多様な意見を分類・整理し、共通点を見出す統合的思考
- **具体例**：ブレインストーミングの結果をMECEに整理し、実行可能な案に絞り込む

### 解説
日常業務での論理的思考の場面を具体的に挙げることで、学習内容と実務の関連性が明確になります。それぞれの場面で求められる思考の種類を理解することで、状況に応じた思考法の使い分けができるようになります。

## 演習1-3：AI指示の改善

### 改善前
「競合他社について調べて」

### 改善後
「当社の主要競合3社（A社、B社、C社）について、以下の項目を調査してください：
1. 直近3年間の売上高推移と市場シェア
2. 主力製品・サービスの特徴と価格帯
3. 最近6ヶ月間の新商品・新サービスの発表内容
4. デジタルマーケティング戦略（SNS活用、Web広告等）
5. 各社の強みと弱みの比較表を作成

調査結果は、経営会議用のプレゼン資料（10枚程度）にまとめてください。」

### 解説のポイント

**改善点1：対象の明確化**
- 「競合他社」→「主要競合3社（具体名）」
- 調査対象を限定し、焦点を絞る

**改善点2：調査項目の具体化**
- 曖昧な「調べて」→5つの具体的な調査項目
- 各項目に具体的な内容を指定

**改善点3：期間・範囲の明示**
- 「直近3年間」「最近6ヶ月間」など時間軸を明確化
- データの範囲を限定して実行可能性を高める

**改善点4：アウトプットの指定**
- 「プレゼン資料（10枚程度）」と成果物を明確化
- 用途（経営会議用）を明示して適切なレベル感を伝える

### 学習ポイント
効果的なAI指示には以下の要素が重要です：
1. **5W1H**の明確化（誰が、何を、いつ、どこで、なぜ、どのように）
2. **具体的な数値**や基準の提示
3. **成果物の形式**の指定
4. **背景や目的**の共有

これらを意識することで、AIから期待通りの結果を得やすくなります。
