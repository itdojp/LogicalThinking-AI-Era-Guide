# AI活用実践ガイド：主要ツール別活用法

## 1. ChatGPT活用ガイド

### 1.1 文書作成での活用

#### ビジネスメール作成
```
【プロンプト例】
以下の内容でビジネスメールを作成してください：
- 宛先：山田部長
- 目的：会議日程の変更依頼
- 変更理由：クライアント訪問が入ったため
- 希望日時：来週火曜日14:00または水曜日10:00
- トーン：丁寧だが簡潔に
```

#### 議事録作成
```
【プロンプト例】
以下のメモから正式な議事録を作成してください：
[会議メモをペースト]
- 日時、参加者、議題を明確に記載
- 決定事項と宿題事項を分けて整理
- 次回会議日程も記載
```

### 1.2 分析・企画での活用

#### SWOT分析
```
【プロンプト例】
当社の新サービス展開についてSWOT分析を行ってください：
- 業界：教育テクノロジー
- サービス：AI学習支援システム
- 競合：A社（シェア30%）、B社（シェア20%）
- 当社の特徴：カスタマイズ性が高い、価格は中程度
```

### 1.3 注意点とベストプラクティス

1. **機密情報の取り扱い**
   - 社名、個人名は仮名に置き換える
   - 具体的な数値は概数に変更

2. **出力の検証**
   - 事実関係は必ず裏取りする
   - 数値計算は再確認する

3. **段階的な指示**
   - 一度に複雑な指示をせず、段階的に精緻化

#### 資料要約
```

1. エグゼクティブサマリー（3行）
```

## 6. トラブルシューティング

### 6.1 期待した出力が得られない場合

#### チェックリスト
- [ ] 指示は具体的か（5W1H）
- [ ] 前提条件は明確か
- [ ] 求める形式は指定したか
- [ ] 例示を含めたか

#### 改善アプローチ
1. **段階的詳細化**
   - まず大枠を作成
   - フィードバックで詳細化

2. **例示の活用**
   - 理想的な出力例を提示
   - 悪い例も示して対比

### 6.2 精度向上のテクニック

#### プロンプトエンジニアリング
1. **役割の明確化**
   ```
   「あなたは経営コンサルタントとして...」
   ```

2. **制約条件の設定**
   ```
   「以下の条件で検討してください：
   - 予算：〇〇万円以内
   - 期間：3ヶ月以内
   - リソース：既存メンバーのみ」
   ```

3. **思考プロセスの指定**
   ```
   「ステップバイステップで考えてください」
   ```

## まとめ：AI活用の成功要因

1. **適材適所**：各AIツールの特性を理解し使い分ける
2. **検証習慣**：出力を鵜呑みにせず必ず検証
3. **継続改善**：プロンプトを保存し、継続的に改良
4. **倫理配慮**：著作権、個人情報、機密情報に注意
